# FAN:STAGE - 팬이 만드는 음악 생태계

> "개성 있는 인디/신인 아티스트들은 자본력 부족으로 유행 따라가기에 급급하고, 진짜 음악 팬들은 자신이 발굴한 아티스트를 효과적으로 응원하고 확산시킬 수 있는 플랫폼이 없다."

FAN:STAGE는 팬이 직접 아티스트를 큐레이팅하고, 그 큐레이션이 다른 팬들에게 자동으로 노출되는 음악 커뮤니티 플랫폼입니다.

## 🎵 주요 특징

- **팬 큐레이션 피드**: 팬들이 만든 플레이리스트가 자동으로 피드에 노출
- **연관 아티스트 자동 추천**: AI 기반 맞춤형 아티스트 발견
- **아티스트 직접 소통**: 아티스트와 팬의 실시간 커뮤니케이션
- **큐레이터 등급 시스템**: 활동에 따른 영향력 향상

## 🚀 실행 방법

1. 프로젝트 클론
```bash
git clone [repository-url]
cd workspace
```

2. 웹 브라우저로 `index.html` 파일 열기
   - 로컬 서버 사용 (권장):
   ```bash
   # Python 3를 사용하는 경우
   python -m http.server 8000
   
   # 또는 Node.js live-server 사용
   npx live-server
   ```

3. 브라우저에서 `http://localhost:8000` 접속

## 📁 파일 구조

```
/workspace
  ├── index.html       # 메인 HTML 파일
  ├── styles.css       # 스타일시트
  ├── script.js        # JavaScript 인터랙션
  └── README.md        # 프로젝트 문서
```

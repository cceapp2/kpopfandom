Problem & Opportunity

인디 음악이 팬을 만나는 방식은 아직 낡았습니다

자본이 없는 아티스트는 유행을 따라가기에 바쁘고, 진짜 팬은 자신이 발굴한 아티스트를 확산할 공간이 없습니다. 우리는 팬이 만드는 열린 음악 생태계를 통해 이 간극을 메웁니다.

아티스트의 페인포인트

  • 마케팅 자본 없이 바이럴을 만들기 어렵습니다.
  • 유행을 좇다 보면 고유한 음악성이 흐려집니다.
  • 대형 플랫폼은 대형 기획사 아티스트 중심입니다.

팬의 페인포인트

  • 응원과 자랑이 닫힌 SNS에 머뭅니다.
  • 알고리즘은 인기곡만 노출합니다.
  • 팬의 활동이 아티스트 성장으로 이어지는 선순환이 없습니다.
One-Liner "인디 아티스트는 팬이 없고, 팬은 확산할 공간이 없다. FAN:STAGE가 그 둘을 연결합니다."

Core Concept

팬이 만드는 열린 음악 생태계

팬의 플레이리스트가 발견 알고리즘이 되고, 연관 아티스트가 자동으로 연결되며, 아티스트는 팬과 직접 소통하는 플랫폼을 만듭니다.

팬 큐레이션 피드

팬이 만든 플레이리스트가 피드에 자동 노출되고, 재생·좋아요가 많을 수록 더 확산됩니다. 팬 등급 시스템으로 영향력까지 보상합니다.

연관 아티스트 추천

내가 좋아한 아티스트와 함께 듣는 다른 신인을 협업 필터링으로 자동 추천. 닫힌 SNS를 넘어선 열린 발견 경험입니다.

아티스트 직접 소통

아티스트가 직접 피드와 댓글로 팬에게 소식을 전하고 피드백을 받습니다. 팬은 활동 데이터를 기반으로 더 깊게 연결됩니다.

MVP Features

12주 안에 구현 가능한 핵심 기능 설계

임팩트, 실현 가능성, 차별성을 고려한 P0-P2 단계별 MVP 로드맵으로 초기 가설 검증에 집중합니다.

P0 · 필수

팬 큐레이션 중심의 음악 경험

  • 음원 업로드와 백그라운드 재생을 지원하는 스트리밍
  • 플레이리스트 생성 시 자동으로 피드 게시
  • 아티스트 피드 및 팔로우 기능
  • 협업 필터링 기반 연관 아티스트 추천
P1 · 성장

팬 활동을 동력으로

  • 씨앗~숲 팬으로 이어지는 등급 시스템
  • 높은 등급 큐레이션 상위 노출 가중치
  • 아티스트와 팬의 댓글·소통 채널
P2 · 확장

지속 가능한 팬덤 경제

  • 팬 펀딩과 스트리밍 연동 후원
  • 라이브 Q&A, 유료 팬 메시지
  • 플랫폼 외부 공유를 통한 확산

12-Week Roadmap

Phase별 목표와 성공 지표

1~6주 핵심 기능 완성, 7~10주 커뮤니티 강화, 11~12주 폴리시 및 배포 준비로 이어지는 실행 가능한 계획입니다.

  • 아티스트 페이지 & 포스트
Week 7-10

Community Features

  • 팬 등급 시스템과 배지
  • 아티스트-팬 커뮤니케이션
